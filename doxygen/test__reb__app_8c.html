Detailed Description

Unit tests for functions in reb/app.c.

This file contains the test suite that validates the main functions of Remote Engine Blocking, in app.c, inicializacion, activation and deactivation of REB and handling CAN messages. It covers success and failure scenarios, as well as different status/byte values.
